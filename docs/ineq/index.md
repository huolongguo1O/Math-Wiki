# 基本的不等关系

## 不等号的性质

### 1. 传递性
若$a \ge b$，$b \ge c$，则$a \ge c$。

### 2. 同向可加性
若$a \ge b$，$c \ge d$，则$a + c \ge b + d$。

### 3. 同乘性
- 若$a \ge b$，$c > 0$，则$ac \ge bc$
- 若$a \ge b$，$c < 0$，则$ac \le bc$

### 4. 乘方性
若$a \ge b > 0$，$n > 0$，则$a^n \ge b^n$。

若$a \ge b > 0$，$n < 0$，则$a^n \le b^n$。

## 重要的不等式
最重要的莫过于\\[ x^2 \ge 0 \\]许多不等式都是从它的基础上推导而来。
