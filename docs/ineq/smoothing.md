# 调整法

## 一、基本原理
调整法是通过逐步调整变量，使不等式趋于等号成立的条件，从而证明不等式的方法。

## 二、线性调整
固定其他变量，调整两个变量，使之相等或趋于某种平衡。

### 示例
设 $a, b, c > 0$，证明：$a^3+b^3+c^3 \ge 3abc$。

**证明：** 不妨设 $a \le b \le c$。若 $a < b$，令 $a' = b' = \frac{a+b}{2}$，$c' = c$。则左边变化：$a^3+b^3 - 2\left( \frac{a+b}{2} \right)^3 = \frac{3}{4}(a+b)(a-b)^2 \ge 0$，所以和不变小。右边：$abc - \left( \frac{a+b}{2} \right)^2 c = -\frac{c}{4}(a-b)^2 \le 0$，所以右边不变大。因此不等式对 $(a', b', c')$ 成立等价于对原式成立。重复调整，最终可使 $a=b=c$，此时不等式取等。所以原不等式成立。

## 三、凹凸性调整
利用函数的凹凸性，通过调整变量使函数值变化。

### 示例
设 $a, b, c > 0$，证明：$\frac{a}{b+c} + \frac{b}{c+a} + \frac{c}{a+b} \ge \frac{3}{2}$。

**证明：** 考虑函数 $f(x) = \frac{x}{1-x}$ 在 $(0,1)$ 上的凸性，通过调整使变量相等。
